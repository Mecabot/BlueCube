Downloads

To sucessfully install BlueCube, you need to install the Arduino Sketch (including the three Arduino Libraries), and at least one of the iOS or Android apps. Once you have all of the components downloaded, please see the Install page for instructions on installing the components.

Arduino Libraries

BlueCube utilises features from my fork of the Cube4 Library. If you have the Freetronics version installed, you will need to replace it with the one below. The new library is a drop in replacement for the Freetronics version.

Version History

  • v1.0.0 (2016-02-XX)
    • Initial Public Release
    • Implemented all API features of the cube Library
    • Added the ability to call user defined functions, and to run previously saved commands (either as static favourites, or from the history)
    • Included colour management features to allow for picking and managing favourite colours
    • Ability to search for multiple cubes, and pick which to connect to
    • App supports Australian and US English, with the appropriate localisation of colour and favourite
